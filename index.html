"ทำปัจจุบันให้ดี — แล้วจะมีอนาคต"

ฉันเป็นนักศึกษาสาขาเทคโนโลยีสารสนเทศที่หลงใหลในการพัฒนาซอฟต์แวร์แบบครบวงจร มีความสามารถทั้งฝั่ง Frontend และ Backend สามารถสร้างระบบที่ใช้งานได้จริงตั้งแต่ UI ไปจนถึง Database

ต้องการหาประสบการณ์ฝึกงาน เพื่อพัฒนาทักษะ และเติบโตในสายงาน Full Stack Development ได้เรียนรู้จากการทำงานจริง ร่วมงานกับทีมที่มีความกระตือรือร้น และมีโอกาสได้สร้างผลงานในอนาคต
